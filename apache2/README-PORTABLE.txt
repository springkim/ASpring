* English *
PORTABLE APACHE 2
Version: 2.2.17 Win32
Author's ICQ: 803-one-45 (spam protect)

Description:
Cut out many unnecessary, build easier but contains all the basic modules and components.
Apache Portable allows you to copy directory APACHE 2 and run it from anywhere.
Unlike simple APACHE 2, now you make the basic configuration using httpd.portable.conf

Features of conf/httpd.portable.conf:
During server start, the file automatically "compiled" in the "correct" httpd.conf and supplied to APACHE 2.
All absolute paths are specified by the variable {$PORTABLEROOT}.
{$PORTABLEROOT} - full path to apache root directory, without endslash.
{$PORTABLEROOT} defined automatically by a small JScript-program.

Startup features:
Launch of portable APACHE 2 occurs by file start.bat, located in the root folder of the server.
Feature of the launch is in the preliminary run wsc-machine to create httpd.conf.

* Русский *
Портативный сервер APACHE 2
Версия: 2.2.17 Win32
ICQ автора: 803-один-45 (защита от спама)

Описание:
Вырезано много лишнего, сборка облегчена и в то же время содержит в себе все основные модули и компоненты.
Портативный апач позволяет вам копировать папку APACHE 2 и запускать её откуда угодно.
В отличии от простого APACHE 2, теперь вы производите основную конфигурацию через файл httpd.portable.conf

Особенности файла conf/httpd.portable.conf:
Во время запуска сервера, файл автоматически "компилируется" в "правильный" httpd.conf и подается APACHE 2.
Все абсолютные пути указываются при помощи переменной {$PORTABLEROOT}.
{$PORTABLEROOT} - полный путь к корневой директории апача, заканчивается без слеша.
{$PORTABLEROOT} определяется автоматически, при помощи небольшой JScript-программы.

Особенности запуска:
Запуск портативного APACHE 2 происходит через файл start.bat, находящийся в корневой папке сервера.
Особенность запуска заключается в предварительном запуске wsc-машины для создания httpd.conf.
